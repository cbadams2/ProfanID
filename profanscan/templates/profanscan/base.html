ProfanID

ProfanID is made by Colin Adams

You can check if a song has explicit lyrics by typing the artist's name and the song's title into the boxes and clicking ProfanScan. If the lyrics can be found, you will either see that explicit words have been identified in your song, or that no explicit language was found. If explicit words are found, the lines containing them will be highlighted and the putative explicit language will be bolded and underlined.

Please note that certain explicit words are often used in compound words and the base explicit word may be found in other words. This search is thorough, and as a result, it may identify "false positives": e.g., words that contain explicit words, but are not themselves explicit.

If lyrics can't be located, or the API yields the wrong result, you will see a page that tells you that the lyrics could not be found.

This website uses the API from musixmatch.com and deezer.com, and as such, is limited by their database.

History of ProfanID

I wrote the original code for ProfanID when I was a freshman in college. I was DJing the 3am slot for my college radio station, which was known as "Safe Harbor" hours, basically when the FCC didn't really care what went out over the air, since all the children were presumed to be asleep by that hour. The prime-time slots, however, were not safe harbor, so explicit language was not permitted to be broadcast. My thinking was that, if I ever did get a prime-time slot, it sure would be nice to have a tool that could help me check if the music I would play was clean or not.. Thus, ProfanID was born.

Around the same time, I was teaching myself to code for the very first time. ProfanID was a great excuse to test things out and do some trial by fire. The first iteration was a monstrosity, as would be expected. And so were the next few after that.

Now, some 8-odd years later, I'm learning to make a website, and the wheel turns back round again. I never did get a prime-time slot: turns out I was an atrocious DJ.. but ProfanID lives on. Hope that it proves useful (or amusing) to you, and if so, maybe consider donating a few $$ to keep the project afloat!

Colin
